# Strata Roadmap

## Phase 1 — Utility First

### Goal:
Make AI tools smarter immediately.

### Deliverables:

- Local MCP server
- Claude memory context
- Cursor personalization
- Skill/domain inference
- Dashboard
- Weekly digest

---

## Phase 2 — Intelligence Layer

### Goal:
Help users improve.

### Deliverables:

- Strength detection
- Workflow optimization suggestions
- Skill acceleration tracking
- Tool benchmarking
- Automation opportunities
- Productivity insights

---

## Phase 3 — Identity Layer

### Goal:
Portable reputation and capability.

### Deliverables:

- Shareable profiles
- Verified growth trajectories
- Capability exports
- Team skill maps
- Professional reputation graph

---

## Phase 4 — Opportunity Layer

### Goal:
Connect talent to opportunity.

### Deliverables:

- Role matching
- Hiring signals
- Internal talent mobility
- Recruiter access (opt-in)
- AI-native career marketplace
