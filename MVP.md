# Strata MVP

## Objective

Validate that users will install and keep Strata because it improves daily AI workflows.

---

## Core Deliverables

### Desktop App

- Tauri desktop shell
- Background process
- Secure local storage

### MCP Server

Endpoints:

- strata://skills
- strata://context/current
- strata://preferences

### Integrations

- Claude
- Cursor

### Dashboard

- Skill map
- Tool usage map
- Strength indicators
- Growth timeline

### Weekly Digest

Examples:

- What you improved at
- What tools you used best
- Suggested optimizations
- Time-saving opportunities

---

## Success Metrics

- 500 installs
- 50% weekly active usage
- 70% say outputs improved
- 60% retain after 30 days
- 40% open weekly digest
